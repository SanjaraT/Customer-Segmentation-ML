--> Overview

This project applies K-Means clustering to segment mall customers based on Age, Annual Income, and Spending Score. PCA is used to visualize clusters in two dimensions.

--> Dataset

Mall_Customers.csv
CustomerID removed
Gender encoded numerically

--> Workflow

Data inspection and basic EDA
Feature scaling using StandardScaler
Optimal cluster selection using Elbow Method
K-Means clustering (k = 5)
PCA-based cluster visualization

--> Tools

Python, Pandas, NumPy, Seaborn, Matplotlib, Scikit-learn
